Increase the visibility and searchability of your products within your Woo Commerce store

== Description ==

The plugin helps to automatically tag and classify your products via image recognition Artificial Intelligence. 247 Labs Automatic Tagger uses cutting edge Artificial Intelligence and Machine Learning Algorithms to identify the objects in your product images and classify (tag) them accordingly. 


== What Is Autotagger? ==

Autotagger is an e-commerce plugin that allows you to instantly add meaningful and precise tags to your products. The days of tedious and time-consuming manual tagging are gone! By using Autotagger, you can quickly add tags to your entire inventory and focus your time to skyrocket your store!

== WHY PROPER TAGGING MAKES A DIFFERENCE? ==
 
Today's consumer makes up to two searches on an e-commerce to find a product, and if they can't find it, they will simply go to another store. In many cases, your store does sell what they are looking for - it was just not searchable!
When you tag your products with terms that your customer uses to search for it, they will find the products they desire and are more likely to purchase them.
 
== DESCRIPTIVE AND SMART TAGGING ==
 
* Autotagger is built with the latest Artificial Intelligence technology to create relevant tags.
* It makes use of image recognition to understand products' attributes and generate multiple tags that in fact describe your products.

== Installation ==

Simply install the plugin, activate in your administration & add your API key in the plugin settings tabs area and  Autotagger will do the rest!

e.g.

1. Download the plugin
2. Create an account and copy an API [here]( http://auto-tagger.thedemo.co/register/ ). 
3. Activate the plugin through the ‘Plugins’ menu in WordPress
4. Go to the Settings page and paste the API code there. The plugin is activated now! 
5. Tag the products
6. See the result!
